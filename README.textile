h1. Notre Dame toolbar extensions

This is a collection of browser extensions.

The purpose is to mimic the ND.edu Popular Sites functionality. I'm starting with the Safari version with plans to add Firefox and Chrome versions.

h2. Use

Click on the "Downloads" link and select the most recent version.
